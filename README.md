# MAPS dataset
MAPS ( MIDI Aligned Piano Sounds ) dataset python api for machine learning  

this api is integrated with:  
1. pytorch/torchvision : multi-core dataloader  
2. mimi/mido : midi file parsing  
3. librosa : data preprocessing
